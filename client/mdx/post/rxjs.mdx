## Introduction

RxJS is a library for composing asynchronous and event-based programs by using
observable sequences.

```js
// @rxjs
const { fromEvents } = rxjs;
fromEvent(document, 'click').subscribe(() => log('clicked!'));
```

### Purity

What makes RxJS powerful is its ability to produce values using pure functions.
